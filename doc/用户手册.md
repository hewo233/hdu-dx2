# 用户手册

## 系统实施手册

后端

### 系统开发环境 

```
Debian GNU/Linux 12 (bookworm) x86_64
```

```bash
go version 1.25.1
psql (PostgreSQL) 17.5
```

前端

```bash
node v20.18.0
npm 11.4.2
```

### 系统部署

后端：默认跑在 8080 端口

```bash
go mod tidy
go run main.go
```

前端：默认跑在 5173 端口

```bash
npm install
npm run serve
```

## 项目结构

后端

```
├── config // 配置文件，默认 git ignore ，需要自行创建 db 和 jwt 配置文件
├── db // 数据库连接、生成
├── doc // 文档
├── handler //项目业务逻辑
├── initall // 初始化
├── middleware // 中间件
├── models // 结构模型
├── route // 路由
├── shared // 一些共享的东西，如常量等
│   └── consts // 常量
└── utils // 工具
    ├── jwt // jwt 相关
    └── password // 密码加密相关
```

前端

```
family-finance-app/
├── src/                    # 源代码目录
│   ├── api/               # API相关文件
│   │   ├── index.ts       # API接口封装
│   │   └── request.ts     # Axios请求配置
│   ├── components/        # Vue组件
│   │   ├── Login.vue      # 登录组件
│   │   └── FamilyManager.vue # 家庭管理主组件
│   ├── pages/            # 页面组件
│   │   ├── LoginPage.vue  # 登录页面
│   │   └── FamilyPage.vue # 家庭管理页面
│   ├── router/           # 路由配置
│   │   └── index.ts      # 路由定义和守卫
│   ├── store/            # 状态管理
│   │   └── auth.ts       # 认证状态管理
│   ├── types/            # TypeScript类型定义
│   │   └── index.ts      # 所有类型定义
│   ├── App.vue           # 根组件
│   └── main.ts           # 应用入口文件
├── public/               # 静态资源
├── package.json          # 项目依赖配置
├── tsconfig.json         # TypeScript配置
├── vite.config.ts        # Vite构建配置
├── index.html            # HTML入口文件
└── README.md            # 项目说明文档
```
